# 🚀 Aplikasi Pengaduan Sarana Sekolah

---

## 📖 Deskripsi

Aplikasi Pengaduan Sarana Sekolah adalah sistem berbasis web yang dirancang untuk mempermudah siswa dalam menyampaikan laporan atau aspirasi terkait sarana dan prasarana sekolah.

Melalui aplikasi ini, proses pengaduan menjadi lebih **efektif, terstruktur, dan transparan**, serta memudahkan admin dalam mengelola dan menindaklanjuti setiap laporan yang masuk.

---

## 🎯 Tujuan

- Mempermudah siswa dalam menyampaikan pengaduan  
- Membantu admin dalam mengelola laporan  
- Menyediakan sistem monitoring status aspirasi  
- Meningkatkan efisiensi penanganan masalah

## 🔄 Alur Sistem

### 🧑‍🎓 Siswa
1. Login ke sistem  
2. Mengisi laporan aspirasi  
3. Mengirim laporan  
4. Melihat status dan histori  
5. Menerima feedback dari admin  

---

### 👨‍💼 Admin
1. Login ke sistem  
2. Melihat data aspirasi  
3. Memproses laporan  
4. Memberikan feedback  
5. Mengubah status (Diproses / Selesai)
